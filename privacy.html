← Runvis 홈으로

개인정보 처리방침

시행일: 2026년 7월 18일 · 버전 1.0

한 문단 요약 — Runvis는 러닝·건강 데이터를 당신의 기기 안에서만 처리합니다. Runvis가 운영하는 서버는 없고, 계정도 없으며, 당신의 운동·건강 데이터를 개발자를 포함한 누구에게도 전송·판매·공유하지 않습니다. 아래에 그 예외 세 가지(날씨 조회, 애플 결제, 지도 경로 생성)를 숨김없이 설명합니다.

1. 수집·처리하는 데이터

Runvis는 다음 데이터를 기기 내에서 읽고 처리합니다. 모두 앱 기능 제공 목적이며, 기기 밖으로 나가지 않습니다.

  • 운동 데이터: 심박수, 페이스, 거리, GPS 경로, 고도, 케이던스, 러닝 파워·보폭·접지시간·수직진폭(지원 기기), 칼로리 — Apple HealthKit을 통해 읽고 저장
  • 건강 데이터: 수면(시간·단계), 심박변이도(HRV), 안정시 심박, VO₂max, 손목 온도 기반 지표, (선택 시) 월경 주기 — 훈련 준비도·회복 분석에 사용
  • 모션 데이터: 손목 가속도(러닝 자동 시작 감지), (착용 시) AirPods 머리 모션 — 폼 신호 참고용
  • 입력 데이터: 목표 대회·기록, 코치 성향, 신발 정보, 운동 후 체크인(강도·통증·수분)
  • Bluetooth 센서: (연결 시) 심박 밴드, 사이클링 파워미터·스마트 트레이너의 실시간 측정값

2. 데이터가 저장되는 곳

  • 운동 기록·경로는 Apple 건강(HealthKit)에 저장됩니다 — 소유자는 당신이며, iCloud 건강 동기화 여부는 iOS 설정을 따릅니다
  • 앱 설정·플랜·코스·배지 계산 결과는 기기 내 앱 저장공간에만 저장됩니다
  • Runvis는 자체 서버·데이터베이스·계정 시스템을 운영하지 않습니다

3. 기기 밖으로 나가는 유일한 예외 3가지

예외전송되는 것받는 곳
날씨 브리핑현재 위치의 대략적 좌표(도시 수준 정밀도)와 시각 — 요청 순간에만, 저장·식별자 없음Open-Meteo(무료 기상 API)
구독 결제결제 처리 전부 — Runvis는 카드·결제 정보를 보지도, 저장하지도 못합니다Apple App Store
코스 자동 생성경로 계산용 출발 좌표 — 생성 요청 순간에만Apple 지도(MapKit)

위 세 경우 모두 이름·이메일 등 식별 정보는 포함되지 않으며, Runvis가 별도로 기록하지 않습니다.

4. 권한 사용 목적

  • 건강(HealthKit): 운동 기록 저장·과거 기록 분석
  • 위치: 러닝 경로 기록, 날씨 브리핑, 코스 안내
  • 모션: 달리기 시작 자동 감지, 준비운동 안내
  • Bluetooth: 외부 센서(심박 밴드·파워미터) 연결
  • 캘린더(선택): 훈련 플랜 일정 추가
  • 사진(선택): 경로 플라이오버 영상 저장

모든 권한은 거부해도 해당 기능 외의 앱 사용에는 지장이 없습니다.

5. 제3자 제공·광고·추적

  • 제3자 제공: 없음
  • 광고 SDK·분석 SDK·추적 도구: 없음
  • 데이터 판매: 없음

6. 데이터 삭제

앱을 삭제하면 앱 저장공간의 데이터가 삭제됩니다. HealthKit에 저장된 운동 기록은 당신 소유이므로 앱 삭제 후에도 건강 앱에 남으며, 건강 앱에서 직접 삭제할 수 있습니다.

7. 아동

Runvis는 만 14세 미만을 대상으로 하지 않으며, 연령 관련 정보를 수집하지 않습니다. 가족 공유 구독으로 자녀가 사용하는 경우 보호자의 지도가 필요합니다.

8. 의료 면책

Runvis의 모든 수치와 안내는 공개된 운동과학 공식에 기반한 참고용 추정치이며, 의학적 진단·치료·처방이 아닙니다. 흉통·어지럼 등 이상 증상이 있으면 즉시 운동을 중단하고 전문가와 상담하세요.
